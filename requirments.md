# Requirements

## MVP (v1.0)
- [ ] Student management (CRUD)
- [ ] Subject organisation (Math, English, etc.)
- [ ] File storage per subject section: Notes, Questions, CATs, Exams
- [ ] Calendar view (FullCalendar)
- [ ] Lesson scheduling (student, subject, date, time, duration)
- [ ] Daily lesson log (what taught, progress %, homework)
- [ ] Student record timeline
- [ ] Automated email sending (scheduled, with attachments)
- [ ] Dashboard (today's lessons, pending emails)

## Future (v2+)
- [ ] Student login/portal
- [ ] Real‑time notifications
- [ ] Mobile app (responsive web first)
- [ ] Pattern detection (struggling topics, best send times)
